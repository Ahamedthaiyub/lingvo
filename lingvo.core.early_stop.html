class lingvo.core.early_stop.EarlyStop(params)[source]¶

Bases: object

Early stopping based on dev-set performance.

Factors out the steps needed to perform early stopping in the trainer when a selected metric hasn’t improved for a given number of steps. If the window param is 0 this is guaranteed to be a no-op.

FProp(theta)[source]¶

Creates an op to determine the best step from the metric history file.

Parameters

theta – Not currently used.

Returns

The created op.

This uses BestStepOp rather than reading the file directly from python in order to ensure compatibility with DevBasedSchedule for learning-rate decay. It is natural to use dev-based decay and early stopping together, for example decaying when dev-set perplexity hasn’t improved for n steps, and stopping when it hasn’t improved for 3n steps.
