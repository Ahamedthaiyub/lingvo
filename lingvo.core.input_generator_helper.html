lingvo.core.input_generator_helper module¶

Helper functions for model.

lingvo.core.input_generator_helper.ComputeSplits(batch_size, num_splits)[source]¶

Creates a tensor of size num_splits of number of values per split.

Assigns each split floor(batch_size/num_splits) and round-robins the remainder (if any) to each split.

Example:

batch_size: [5]
num_splits: 3
returns: [2, 2, 1]
Parameters

  • batch_size – tensor of rank 0, size of tensor to be split

  • num_splits – number of splits to split tensor into

Returns

tensor of length num_splits containing sizes of each split

lingvo.core.input_generator_helper.SplitTensors(xs, num_splits)[source]¶

Splits tensors in xs evenly into num_splits along the 1st dimenion.

Parameters

  • xs – A tuple of tensors. Each tensor’s 1st dimension is the same size.

  • num_splits – A python integer.

Returns

A tuple of lists of tensors, num elements in the tuple = len(xs).

i-th element in each list corresponds to i-th split of each tensor in xs along the first dimension of each tensor.

lingvo.core.input_generator_helper.SplitDictOfTensors(t_dict, num_splits)[source]¶

Splits tensors in t_dict evenly into num_splits along the 1st dimenion.

Parameters

  • t_dict – A dictionary of tensors. Each tensor’s 1st dimension is the same size.

  • num_splits – A python integer.

Returns

A list of dictionaries of tensors, num elements in the list = num_splits

i-th dictionary in the list corresponds to i-th split of each tensor along the first dimension of each tensor for each key in the original dict.
