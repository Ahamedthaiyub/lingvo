lingvo.core.spectrum_augmenter_on_device module¶

Lingvo layers that are used for spectrum augmentation on-device.

class lingvo.core.spectrum_augmenter_on_device.SpectrumAugmenterOnDevice(*args, **kwargs)[source]¶

Bases: lingvo.core.spectrum_augmenter.SpectrumAugmenter

Performs data augmentation as according to the SpecAug paper.

This implementation uses portable replacements for the tf.einsum ops.

https://arxiv.org/pdf/1904.08779.pdf
