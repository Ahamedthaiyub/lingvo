lingvo.ipython_kernel module¶

Starts an IPython kernel with Lingvo deps.

jupyter_http_over_ws must be installed and activated. See https://research.google.com/colaboratory/local-runtimes.html for more details.

To use:

bazel run -c opt //lingvo:ipython_kernel
