lingvo.model_registry module

Convenience helpers for managing Params for datasets and models.

Typical usage will be to define and register a subclass of ModelParams for each dataset.

classmethod _ModelParamsClassKey(src_cls)[source]

Returns a string key used for src_cls in the model registry.

The returned key is a period separated string. E.g., image.mnist.LeNet5. It roughly reflects how params files are organized. We put some of the directory information into the key to avoid future model name conflicts.

Parameters

src_cls – A subclass of BaseModel.
