_ProcessLine(line)[source]

A single-text-line processor.

Gets a string tensor representing a line of text that have been read from the input file, and splits it to graphemes (characters). We use original characters as the target labels, and the lowercased and punctuation-removed characters as the source labels.

Parameters

line – a 1D string tensor.

Returns

A list of tensors, in the expected order by __init__.
